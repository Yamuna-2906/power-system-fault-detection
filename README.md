# Power System Fault Detection and Classification

IBM SkillsBuild AICTE Internship Project (PS 41)

## Overview
Built using IBM Watson Studio AutoAI (no-code) to classify power system faults 
(Line Breakage, Transformer Failure, Overheating) using electrical measurement data.

## Tech Stack
IBM Cloud Lite, Watson Studio, AutoAI, watsonx.ai Runtime

## Model
Snap Logistic Regression (Accuracy: 0.398)

## Dataset
https://www.kaggle.com/datasets/ziya07/power-system-faults-dataset
